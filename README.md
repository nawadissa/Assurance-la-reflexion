Bienvenue sur le site web de **Assurance La Réflexion**. C’est un site statique simple présentant les services proposés par l'entreprise.

## Description
Ce site web est conçu pour informer les clients sur les différents services d'assurance et de mobilité offerts par l'entreprise "Assurance La Réflexion". Il s'agit d'une interface simple et épurée permettant une navigation rapide.

## Services proposés
Le site met en avant les services suivants :
- **Assurance Auto** : Protection de votre véhicule pour rouler en toute tranquillité.
- **Auto-école** : Préparation à l'examen du permis de conduire avec des experts.
- **Assurance Immobilière** : Sécurisation de votre maison contre les imprévus.
- **Vente et location autos** : Services de confiance pour la vente et la location de véhicules.

## Structure du site
Le site est composé de plusieurs pages (accessibles via le menu de navigation) :
- `Accueil`
- `Services`
- `À propos`
- `Contact`

## Technologies utilisées
- HTML5
- CSS3 (pour la mise en forme)
- JavaScript
